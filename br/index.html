Cliprz framework

O Cliprz é um programa framework PHP gratuito em código aberto escrito em PHP versão 5.3.0 ou posterior. Ele te ajudará a criar um projeto PHP sem precisar inserir códigos grandes. O que distingue Cliprz Framework é que ele é fácil, rápido, simples e seguro. Nele você pode rotear URLs sem usar controlador/funções/parâmetros. Além disso, você pode usar controlador de bancos de dados MySQL ( o fácil sistema MySQL). Um jeito fácil para criar um rápido painel administrador (scaffold), sistema com páginas de controle completa. E ainda pode te ajudar com plugins, biblioteca e muito mais!
